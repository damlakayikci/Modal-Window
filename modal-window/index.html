Here is a very small project to understand how JavaScript works with click event.
All these three modals are connected to the same text, so we use 'document.querySelectorAll' to select all of them.
To close the modal just click the '×' icon or anywhere out of the modal. Or you can simply press the key 'esc'.
In this project, 'classList.add', 'classList.remove' and 'classList.contains' methods are used to add and remove the 'hidden' class in the CSS file which you can find in the 'script.js' file. You can also find detailed notes about certain methods in the file.
